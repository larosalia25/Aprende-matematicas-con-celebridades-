Suceciones Númericas


Suseción de patrón simple

como ya lo hemos visto las suceciones númericas son aquel conjunto de números formados por un cierto patrón, este tipo de suseción es una suseción cuyo patrón solo se define por suma por ejemplo: 2, 4, 6, 8, 10, 12.

Suseción con distintas operaciones

Este tipo de suceción cuenta con un solo patrón pero con diferentes operaciones a suma y reta puede ser multiplicando o dividiendo por ejemplo: 2,2,4,12,48 en está sucecion comenzamos con el elemento 2, este se multiplicara por 1 para obtener el siguiente elemento, 2x1=2, ahora el segundo elemento se multiplica por dos 2x2=4, y así aumenta un número con cada nuevo elemento que a la vez se multiplicara por el elemento anterior.

Suseción con diferencia

Este tipo de suceción solo tiene un patrón pero este no suma si no que resta, es muy sencecillo ya que solo cuenta con un patrón pero este en regresivo, por ejemplo: 100, 90, 80, 70...

Suseción combinada

Este tipo de suceción tiene dos patrones esto se logra con diferentes operaciones en la misma suceción por ejemplo 10, 5, 15, 7.5, 22.5... en está sucecion primero se divide entre dos y luego se multiplica por 3, haciendo un doble patron utilizando diferentes operaciones.
